# Excel-Data-Weekly-Project-Customer-Segmentation-and-Data-Profiling

In this project you will practice the concepts learnt in first week of - Data Analysis 
 - NAVIGATION WITHIN EXCEL 
 - FUNCTIONS AND FORMULAS
 - CONDITIONAL FORMATTING AND FILTERS 
 - DATA MANAGEMENT (LOOK-UPS AND REFERENCING)
 - CONDTIONAL AGGREGATION 
 
**Objective**

In this project you will clean data provided using Excel functions and provide insights in terms of transaction volume and times on transactions of a simulated client situation. 

[Dataset_Link](https://drive.google.com/file/d/1GD15KgjfLBotCCDd2IjNg2bR_wuXnKAd/view?usp=sharing)

**Instructions**

The task is based on a synthesized transaction dataset for 100 hypothetical customers , the dataset is designed to simulate realistic transaction behaviour and insights you gain from the task will be genuine
Steps to take:
- Load the data in a tool of your choice (R, Excel, Python, Tableau) most preferably Excel as it is the main focus of this week. 
- Start by doing basic checks on the data issues, write an email to the client (hypothetical) , highlighting the issues and how they will affect your analysis going forward, and provide recommendations for future improvements in the data collection stage .
- Gather some interesting overall insights about the data , eg monthly transactions by customers , average transaction amounts, etc 
- Segment the dataset by transaction  date and time , visualize transaction volume and spending over the course of an average day 
- What insights can you draw from the provided dataset? 
- Put together 2-3 slides summarizing the interesting insights you find in the dataset. 

**Deliverables**
1. Email to client highlighting issues and recommendations to avoid this in the future. 
2. 2-3 slides summarizing the interesting insights 

Upload these to your github repository for Technical Mentor review.
